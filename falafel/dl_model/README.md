# DL_model


This subfolder is independent to other code (besides falafel.conf file).

It allows :
1. getDatabse.py :   The extraction of database from a zip.
2. createModel.py :  The creation of a DL model, using keras API
3. trainModel.py :   The training of the model. All history of a certain model are saved in model directory (./model)
4. prediction.py :   The analyses of few sample files. The result is stored in logs folder in main repo.

The resulting model is stored in the model directory (./model).
